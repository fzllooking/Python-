# python中的注释
* 单行注释：使用一个# ,例如：
```
# 这是一行注释
```

* 多行注释：使用一对 3个双引号"""   """ ,例如：
```
"""
我是
注释
"""
```
* 注意，如果使用变量取接收多行注释，那么其中的内容就为字符串：
```
name = """我叫小明，你叫什么"""
```

# 变量的用法
* 在print中，可以有：
```
money = 50
print("钱包还有:" , money)
#结果：钱包还有50
```

```
money=money-10
print("买了冰淇淋花费10元，还剩余:"，money，"元")
#买了冰淇淋花费10元,还剩余40元
```
